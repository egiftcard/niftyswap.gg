## Run Locally

Clone the project

```bash
  git clone https://github.com/nkrishang/niftyswap.gg.git
```

Go to the project directory

```bash
  cd niftyswap.gg
```

Install dependencies

```bash
  yarn install
```

Run tests

```bash
  npx hardhat test
```
